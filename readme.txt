A Request for Quotation (RFQ) module for MarketKing, compatible with HPOS.

== Description ==

This plugin allows buyers to create RFQs and suppliers to respond with quotations. It is fully compatible with WooCommerce's High-Performance Order Storage (HPOS).

== Installation ==

1. Upload the `marketking-rfq` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the `[rfq_form]` shortcode to display the RFQ form on any page.

== Frequently Asked Questions ==

= How do I create an RFQ? =

As a buyer, use the `[rfq_form]` shortcode on any page to create a new RFQ.

= How do suppliers respond to RFQs? =

Suppliers will receive notifications and can view RFQs in their dashboard.
